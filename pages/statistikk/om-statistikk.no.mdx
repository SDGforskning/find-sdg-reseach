# Om statistikk

Disse sidene inneholder statistikk om forskningspublikasjoner knyttet til bærekraftsmålene, både på nasjonalt (Norge) og institusjonelt-nivå. 
Statistikken er basert på den breddeste kartleggingen, se [Om denne tjenesten](/om/om-tjenesten#metoder-for-tilknytning-til-b%C3%A6rekraftsm%C3%A5lene) for informasjon.
